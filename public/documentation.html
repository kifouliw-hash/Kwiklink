Documentation API KwikLink – V2

L’API KwikLink permet aux développeurs d’intégrer des paiements unifiés — Mobile Money, cartes bancaires et portefeuilles KwikSend — sur leurs sites et applications. Elle est sécurisée, rapide et compatible avec tous les environnements modernes.

Authentification

Chaque requête doit inclure la clé API secrète :

Authorization: Bearer sk_test_abc123
Content-Type: application/json

Clés disponibles dans le Dashboard → Développeurs > Clés API

Environnements

Sandbox : https://sandbox-api.kwiklink.com
Production : https://api.kwiklink.com

Créer un paiement

POST /v1/payment_intents
{
  "amount": 5000,
  "currency": "XOF",
  "method": "mobile_money",
  "customer": "client_001",
  "description": "Achat produits marché Adjamé"
}

Réponse :

{
  "status": "success",
  "transaction_id": "kl_48392A",
  "redirect_url": "https://checkout.kwiklink.com/kl_48392A"
}

Gestion de portefeuille

SDKs & intégrations

Node.js

const kwiklink = require('kwiklink')('sk_test_abc123');
await kwiklink.paymentIntents.create({
  amount: 5000,
  currency: 'XOF',
  method: 'mobile_money'
});

Python

import kwiklink
kwiklink.api_key = "sk_test_abc123"
kwiklink.Payment.create(amount=5000, currency="XOF")

PHP

$kwik = new KwikLink("sk_test_abc123");
$kwik->createPayment([
  "amount" => 5000,
  "currency" => "XOF"
]);

cURL

curl -X POST https://api.kwiklink.com/v1/payment_intents \
-H "Authorization: Bearer sk_test_abc123" \
-d '{"amount":5000,"currency":"XOF","method":"mobile_money"}'
